# emoji prefix

- 🐛  `:bug:` バグ修正
- 👍  `:+1:` 機能改善
- 🎉  `:tada:` 機能追加
- 🚧  `:construction:` WIP
- ♻️   `:recycle:` リファクタリング
- 🚿  `:shower:` 不要な機能・使われなくなった機能の削除
- 💚  `:green_heart:` テストやCIの修正・改善
- 👕  `:shirt:` Lintエラーの修正やコードスタイルの修正
- 🚀  `:rocket:` パフォーマンス改善
- ⬆️   `:arrow_up:` 依存パッケージなどのアップデート
- ⬇️   `:arrow_down:` 依存パッケージのダウングレード
- 🔒  `:lock:` 新機能の公開範囲の制限
- 👮  `:cop:` セキュリティ関連の改善
